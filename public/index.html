Direction

多種多様な人間と関わってきた荒川 友人がチームに加わることで、
コミュニケーションが円滑になり、チーム全体の生産性・品質が向上します。

  • コミュニケーション
    チーム内外・職種を問わず、コミュニケーション能力を発揮します。
  • リーダーシップ
    率先してチーム・プロジェクトを
    成功に導きます。
  • 情報整理
    議事録・マニュアルを整備し、
    チームの生産性を向上させます。
  • 要件定義
    要望と目的のヒアリングの上、 提案と判断をしながら要件を明確にします。
  • ワイヤーフレーム
    実現性の高さ・パターンが想定された、ワイヤーフレームを作成します。
  • 進行管理
    ガントチャート・カンバンなどを活用の上、最適な稼働状況を実現します。

荒川 友人の加入は、
組織の発展に繋がります。

荒川 友人の制作環境構築スタイルは、組織・個人の成長を促します。
レガシーな環境を一新し、トレンド技術を導入。段階的にルールを整備し、「この職場で何が出来るか？」を明確に。
グラフは過去に在籍したチームの事例です。3ヶ月ごとの人員数を指標とした組織の発展推移になります。
積み上げた実績の結果、クライアント側より委託側チームの増員要請※を頂けました。

※2017/03時点で荒川管轄の業務委託チームを最大7名体制とのご要望。
  • 入社時人員数
    2人
  • 1年半後の人員数
    13人
  • 人員増加倍率
    6.5倍

Cording

荒川 友人はコーダーとしてもアサイン出来るWEBディレクターです。
エンジニアが関わるシステム要件に定評があります。

  • HTML
    レイアウト・文章構造に基いた、セマンティックなコードを提供します。
  • CSS
    プロジェクトに最適なcss設計で構築。
    レスポンシブ・sassにも対応します。
  • Javascript
    プラグインの実装・カスタマイズによってリッチな演出を提供します。
  • CMS
    WordPress・MovableTypeのテンプレートを要件に合わせて作成します。
  • タスクランナー・ビルドツール
    gulpやwebpackを使用し、
    効率的な作業環境を構築します。
  • バージョン管理
    gitを使用し、git flowをベースに最適なソースコードのやり取りを行います。

Plan

荒川 友人のチーム加入をご検討されるにあたっての、
3つのモデルケースをご紹介します。

  • Full Performance

    • ディレクションとコーディングの兼務に加えて、組織単位のチームマネージメントを行います。
    • 荒川 友人のパフォーマンスを最も活かせるプランです。
      チームにコミットした上で、実務を通して課題に対して施策。そして解決。
      生産性・品質を向上させ、組織全体を発展させます。
    • 推奨雇用形態：正社員・契約社員

  • Director

    • ディレクションを主軸に、コーディング業務もサポートします。
    • ディレクションとコーディングの業務比率は8:2が基準です。
      プロジェクト単位でのアサインを推奨します。
    • 推奨雇用形態：派遣社員・業務委託(準委任)

  • Corder

    • コーディングを主軸に、デザイン業務もサポートします。
    • コーディングとデザインの業務比率は8:2が基準です。
      要件に応じて、御社のデザイナーのサポートを行いながらコーディングを行います。
    • 推奨雇用形態：派遣社員・業務委託(準委任/請負)
  • Full Performance
  • Director
  • Corder
